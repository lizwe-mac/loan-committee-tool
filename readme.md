# Loan Committe Tool

A simple app for rating loan applicants

## Stack

MERN, MaterialUI, JWT

## How to run app

Create .env file and add your MONGO_URI

### Install dependencies

```
# Backend dependencies
npm install

# Frontend dependencies
cd frontend
npm install
```

### Run Server & Client

```
# Use concurrently - npm module
npm run dev
```
